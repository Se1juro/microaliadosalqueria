# Gestion de inventario microaliados Alqueria
Sistema de gestion de inventario para microaliados de Alqueria.

Este sistema esta dise√±ado para los clientes que anteriormente organizaban sus inventarios a papel y lapiz puedan tener una gestion mas amigable, rapida y eficaz a la hora de consultar y modificar su inventario.

# HOW TO USE

1. npm install
2. npm start | npm run dev

# REQUIREMENTS

1. [NodeJS](https://nodejs.org/en/)
2. NPM
3. [MongoDB](https://www.mongodb.com/try/download/community?tck=docs_server)
